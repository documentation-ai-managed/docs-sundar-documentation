## Getting Started with UXPin

UXPin is a collaborative UX design platform that enables product teams to build interactive prototypes without coding. It uses real code components and AI-driven automation to streamline design workflows, accelerate design systems, and bring consistency to products. Used by global teams at top companies like Microsoft and Amazon, UXPin bridges the gap between design and development, allowing for production-ready prototypes with the power of code components and design systems.
